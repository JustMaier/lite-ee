lite-event-emitter
==================

An ultralite es6 event emitter with on, once, off, and emit

```
npm i --save git+https://github.com/justmaier/lite-event-emitter.git
```

- `on('event', (...args) => {})`
- `once('event', (...args) => {})`
- `off('event'[, handler])`
- `emit('event', ...args)`
